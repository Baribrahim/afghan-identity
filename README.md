# Maintaining Afghan Identity While Living Abroad

An IB Personal Project journal website exploring cultural identity and heritage preservation.

## Project Overview

This website serves as a personal journal documenting the journey of understanding and maintaining Afghan cultural identity while living abroad. It represents the "Product" component of an IB Personal Project.

**Learning Goal:** Developing a thorough and meaningful understanding of my cultural background, especially the Afghan origins of my ancestors is my learning objective for this personal project.

## Website Structure

The website is organized into 10 main sections, each corresponding to a research task:

1. **Task 1:** Define Your Personal Motivation
2. **Task 2:** Explore Afghan Identity in Your Own Life
3. **Task 3:** Research Family as the Carrier of Culture
4. **Task 4:** Study Language and Cultural Connection
5. **Task 5:** Document Food as Cultural Memory
6. **Task 6:** Examine Traditions and Celebrations Abroad
7. **Task 7:** Identify Core Afghan Values in Daily Life
8. **Task 8:** Analyse Challenges of Living Between Cultures
9. **Task 9:** Propose Ways to Preserve Afghan Identity
10. **Task 10:** Reflect on Your Learning & Growth

## Setting Up GitHub Pages

### Method 1: Using GitHub Web Interface

1. Create a new repository on GitHub
2. Upload all files (index.html, styles.css, script.js, README.md)
3. Go to repository Settings → Pages
4. Select the main branch as source
5. Your site will be available at: `https://[username].github.io/[repository-name]`

### Method 2: Using Git Command Line

```bash
# Initialize git repository
git init

# Add all files
git add .

# Commit files
git commit -m "Initial commit: IB Personal Project website"

# Add remote repository (replace with your repository URL)
git remote add origin https://github.com/[username]/[repository-name].git

# Push to GitHub
git branch -M main
git push -u origin main
```

Then enable GitHub Pages in repository settings.

## Customization

### Updating Content

All content is in `index.html`. Simply edit the text within each section to replace example content with your own experiences and reflections.

### Changing Colors

Edit the CSS variables in `styles.css` at the top of the file:

```css
:root {
    --primary-color: #1a1a2e;
    --secondary-color: #16213e;
    --accent-color: #e94560;
    /* ... other colors ... */
}
```

### Adding Images

To add images:
1. Create an `images` folder
2. Add your image files
3. Reference them in HTML: `<img src="images/your-image.jpg" alt="Description">`
